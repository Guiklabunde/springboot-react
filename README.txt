# Aplicação de Gerenciamento de Produtos

Este é um projeto de exemplo que demonstra como criar uma aplicação de gerenciamento de produtos com uma API em Spring Boot no backend e o React no frontend.

# Requisitos

- [Node.js](https://nodejs.org/) para executar o aplicativo React.
- [Java](https://www.oracle.com/java/technologies/javase-downloads.html) para executar a API Spring Boot.
- Um servidor de banco de dados (como MySQL, PostgreSQL, etc.) configurado e em execução.

# Configuração

# Configuração da API Spring Boot

1. Clone o repositório do projeto:

   git clone https://github.com/Guiklabunde/produtos.git
   cd produtos/backend

Abra o projeto em sua IDE favorita.

A API será executada em http://localhost:8080.

# Configuração da Aplicação React

1. Abra o repositório da aplicação React:

Instale as dependências:

npm install

Edite o arquivo App.js para configurar a URL da API, caso seja diferente de http://localhost:8080.

# Inicie o aplicativo React:

npm start

O aplicativo React será executado em http://localhost:3000.

# Funcionalidades:

 Listar produtos a partir da API.
 Adicionar um novo produto.
 Editar um produto existente.
 Remover um produto.

-> Uso:

Acesse o aplicativo React em seu navegador em http://localhost:3000.

Você pode listar, adicionar, editar e remover produtos.
